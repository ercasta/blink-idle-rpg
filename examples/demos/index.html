üéÆ About Blink Idle RPG

Blink is an idle RPG where you define your party and strategy, then watch the entire game unfold automatically. The system uses:

  • BRL (Blink Rule Language): Define game rules and mechanics
  • BCL (Blink Choice Language): Set player strategies
  • Multiple Engines: JavaScript, Rust, and Batch engines

üì¶ All demos now run from actual BRL/IR files! Download them from each demo page to study or modify the game rules and strategies.
