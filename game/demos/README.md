# Blink Idle RPG - Demo Package

This directory contains standalone browser demos for the Blink Idle RPG engine.

## Contents

- `index.html` - Demo launcher listing available demos
- `rpg-demo.html` - Classic RPG demo using the Blink Engine
- `ir/` - Pre-compiled IR files (auto-generated, not committed)
- `blink-engine.bundle.js` - Blink Engine (auto-generated, not committed)
- `blink-compiler.bundle.js` - BRL Compiler (auto-generated, not committed)

## Running the Demos

The JS bundles and IR files are generated by the build process and are not included in source control.

Build everything:

```bash
# From repository root
make dev-setup     # Build TypeScript packages and bundles
make compile-brl   # Compile BRL source to IR (including demo IR)
```

Then serve the demos using a local web server:

```bash
cd game/demos
npx serve .
```

Or using Python:

```bash
cd game/demos
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## How It Works

The demo loads pre-compiled IR (Intermediate Representation) files that contain
all game rules, components, and initial entity state. The BRL source files in
`../brl/` are compiled to IR at build time by `tools/compile-brl-to-ir.js`.

This approach means:
- No BRL compiler is needed at runtime in the browser
- Faster page load (no compilation step)
- Smaller payload (no compiler bundle required for the demo)
