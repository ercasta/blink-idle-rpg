# Blink Idle RPG - Demo Package

This directory contains standalone browser demos for the Blink Idle RPG engine.

## Contents

- `index.html` - Demo launcher listing available demos
- `rpg-demo.html` - Classic RPG demo using the Blink Engine
- `blink-engine.bundle.js` - Blink Engine (auto-generated, not committed)
- `blink-compiler.bundle.js` - BRL Compiler (auto-generated, not committed)

## Running the Demos

The JS bundles are generated by the build process and are not included in source control. To build them:

```bash
make dev-setup
```

Then serve the demos using a local web server:

```bash
cd game/demos
npx serve .
```

Or using Python:

```bash
cd game/demos
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

## BRL Files

The game rules are written in BRL (Blink Rule Language) and located in `../brl/`.
The demos load and compile these rules in-browser using the TypeScript compiler bundle.
