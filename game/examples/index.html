🎮 Blink Idle RPG

Engine test examples — see game/examples/README.md

⚔️ Play the Game

The full game runs as a React web app powered by a Rust/WASM engine. Build and start it with:

npm run setup && npm run build:wasm && npm run install:wasm && npm run dev:app

Or visit the live deployment on GitHub Pages.

🔧 Test the Engine

The BRL programs in brl/ are small standalone examples that validate the BRL → Rust compilation pipeline. Run them with:

npm run test:harness

Tests: counter (scheduling), combat (entities, damage, targeting), functions (user-defined functions & builtins), classic-rpg (full game rules).
